For landing and UI work, preserve existing brand assets, keep the page directly usable without a build step when the folder is static, and verify key responsive widths before finalizing.

Use visual hierarchy, spacing, contrast, and concise copy to make the first viewport communicate the product's value without requiring the user to read the whole page.
